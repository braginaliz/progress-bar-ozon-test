
# Блок прогресс для web-приложений

Это прототип блока прогресса, каторый можно использовать в своих проектах. Он умеет показывать сколько прогресса выполнилось (например загрузка файла), а также может крутиться и прятаться.

## Демо

Посмотреть в действии можно тут: https://braginaliz.github.io/progress-bar-ozon/

## Чо он умеет?

- **Normal** – обычное состояние. Через поле Value пользователь может задать число от 0 до 100, и синяя дуга растёт по часовой стрелке, начиная с 12 часов.
- **Animate** – если включить этот переключатель, весь блок начинает равномерно вращаться по часовой стрелке. Анимация зависима от значения (для проверки рекоммендуется установить значения от 1 до 99).
- **Hide** – при включении блок плавно исчезает (становится прозрачным и не реагирует на нажатия). 

## Как подключить к своей странице

1. Скопируйте файлы: `index.html`, `styles/` (всю папку), `script.js` и `app.js`.
2. В HTML должен быть контейнер с классом `container` 
   Внутри контейнера обязательны элементы с атрибутами:
   - `data-progress-wrapper` – обёртка для всего блока.
   - `data-progress-circle` – сам свг-элемент.
   - `data-progress-bar` – круг, который отвечает за дугу прогресса (второй `<circle>`).
3. Подключите скрипты и стили.
4. Создайте экземпляр класса `Progress`, передав селектор контейнера или сам DOM-элемент.


## API 

У экземпляра класса есть методы:

- `setValue(value)` – устанавливает значение прогресса (целое число от 0 до 100). Автоматически обрезает некорректные значения.
- `setAnimated(boolean)` – включает (`true`) или выключает (`false`) анимацию вращения.
- `setHidden(boolean)` – показывает (`false`) или скрывает (`true`) блок.
- `getValue()` – возвращает текущее значение.

## Адаптивность

На телефонах в портретной ориентации блок выглядит вертикально, как в макете. Если повернуть телефон в альбомный режим, элементы перестраиваются в строку.
